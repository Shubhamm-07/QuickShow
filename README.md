# Quick Show - Movie Ticket Booking Web App

A modern, responsive movie ticket booking web application that allows users to browse movies, select showtimes, choose seats, and book tickets online.

## üé¨ Live Demo
**[View Live Project](https://quick-show-eight.vercel.app)**

## ‚ú® Features

- **Movie Browsing**: Browse current movies with details, ratings, and trailers
- **Showtime Selection**: View available showtimes for different theaters
- **Seat Selection**: Interactive seat map with real-time availability
- **Secure Payment**: Integrated payment gateway for seamless transactions
- **Email Notifications**: Automated booking confirmations and show reminders
- **Smart Reminders**: Countdown email sent 8 hours before showtime with movie details
- **New Show Alerts**: Email notifications for newly added movies and shows
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Clerk Authentication**: Seamless user registration and login with Clerk
- **Booking Management**: View and manage your ticket bookings

## üõ†Ô∏è Technologies Used

- **Frontend**: React.js 
- **Styling**: CSS3 / Tailwind CSS
- **Deployment**: Vercel
- **State Management**: Context API 
- **Authentication**: Clerk (User Management & Authentication)
- **Database**: MongoDB 
- **Payment Integration**: Stripe
- **Email Service**: NodeMailer 
- **Scheduling**: Node-cron
